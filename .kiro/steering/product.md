# Product Overview

Smiths Detection E-Commerce Platform - A web-based platform for browsing and purchasing detection equipment products.

## Current Status

- Backend: 60% complete (API endpoints, services, database)
- Frontend: 10% complete (basic React setup)
- Overall: 35% complete

## Core Features

- Product catalog browsing (74 products from CSV)
- Product detail pages with reviews
- Shopping cart management (add, update, remove items)
- Persistent cart storage in MySQL
- CSV-based product import on server startup

## Key Characteristics

- Single-page application (React)
- RESTful API design
- Single-server deployment model
- No authentication/checkout (phase 1 scope)
- Emoji icons for product visualization
